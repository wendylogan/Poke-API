# Pokémon API Viewer  

A simple web app that fetches and displays Pokémon data (name, height, weight) from the [PokéAPI](https://pokeapi.co/).  

## Features  
- Dropdown menu to select a Pokémon.  
- Displays the selected Pokémon's **name**, **height**, and **weight**.
